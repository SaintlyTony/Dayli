# Dayli / NutriGo

NutriGo — сервис доставки здорового питания. Мобильное приложение рассчитывает рацион под цели пользователя и позволяет оформить доставку.

## Функционал

- Анкета с расчётом суточных калорий и БЖУ (учёт ограничений и аллергий)
- Нижнее меню: «Рацион», «Прогресс», «Доставка», «Профиль»
- Экран "Рацион" получает меню с сервера
- Экран "Прогресс" хранит локальный журнал веса
- Экран "Доставка" позволяет оформлять заказы и видеть историю

## Структура репозитория

- `mobile/` — мобильное приложение React Native
- `admin/` — веб‑панель для управления меню и заказами
- `backend/` — сервер на Node.js c PostgreSQL

## Требования

- Node.js и npm
- React Native CLI или Expo
- PostgreSQL для backend

## Установка

```bash
git clone <repo-url>
cd Dayli
# мобильное приложение
cd mobile && npm install && cd ..
# сервер
cd backend && npm install && cd ..
```

## Запуск

```bash
# мобильная часть
cd mobile && npm start
# далее npm run android или npm run ios
```

В другом терминале запустите сервер:

```bash
cd backend && npm start
```

На Android-эмуляторе сервер доступен по адресу `http://10.0.2.2:3000`.

## Тесты

Во всех частях репозитория используются Jest-тесты. Запустить их можно командой `npm test` в каталогах `mobile` и `backend`.
