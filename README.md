# Dayli / NutriGo

NutriGo — сервис доставки здорового питания. Приложение помогает рассчитать рацион под ваши цели: похудение, набор массы или поддержание формы. Сейчас реализован базовый экран анкеты, рассчитывающий суточную калорийность и БЖУ.

## Структура репозитория

- `mobile/` — код мобильного приложения на React Native.
- `admin/` — панель администратора для управления меню и заказами.
- `backend/` — серверная часть (API и база данных).

## Требования

- Node.js и npm/yarn
- React Native CLI или Expo

## Установка

```bash
# клонировать репозиторий
git clone <repo-url>
cd Dayli
# установить зависимости мобильного приложения
cd mobile
npm install
```

## Запуск

```bash
# запустить metro bundler
npm start
# далее собрать приложение
npm run android   # или npm run ios
```
