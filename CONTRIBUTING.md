# Contributing to NutriGo

Спасибо за интерес к проекту! Прежде чем отправлять изменения, выполните следующие шаги:

1. Форкните репозиторий и создайте ветку для своей задачи.
2. Установите зависимости в каталогах `mobile` и `backend` с помощью `npm install`.
3. Запустите `npm test` в каждом каталоге и убедитесь, что все тесты проходят.
4. Оформляйте коммиты осмысленными сообщениями на английском.
5. При создании pull request опишите сделанные изменения и ссылку на задачу, если она есть.

При возникновении вопросов создавайте issue или обсуждение.
