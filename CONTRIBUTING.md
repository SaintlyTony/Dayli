# Contributing to NutriGo

Спасибо за проявленный интерес к проекту! Для предложения изменений создайте форк репозитория и отправьте pull request. Постарайтесь следовать этим правилам:

- Используйте осмысленные сообщения коммитов.
- Пишите код по возможностям на английском, комментарии — на русском или английском.
- Перед отправкой убедитесь, что проект собирается и проходят тесты (если они есть).

Если у вас есть вопросы, создайте issue или напишите в обсуждении проекта.
